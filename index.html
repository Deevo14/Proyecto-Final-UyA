¿Qué es Organize?

Organize es una herramienta con la cual tienes la posibilidad de crear tu propio horario de una forma intuitiva y rápida, pudiendo organizar tus tiempos de estudio. Además podrás ver un seguimiento de si has cumplido el horario creado y ver tus avances.

Características de Organize

En Organize estamos siempre desarrollando nuevas funciones. Ahora mismo solo tenemos disponible para el usuario el poder crear un horario añadiendo nuevas tareas


Pero nuestro objetivo final es mucho más que estas simples tres funciones. Tenemos como finalidad una lista de funcionalidades a desarrollar:

  • Realizar horario.
  • Calendario para seguimiento del horario.
  • Función de "empezar a trabajar" para seguimiento del trabajo.
  • Función de "trabajo realizado" para finalizar el seguimiento.
  • Mantener un seguimiento del tiempo de ocio y de trabajo para optimización de este.
  • Vincular la cuenta de la ULL.
  • Agregar Tareas del Campus virtual.
  • Calendario con los proximos eventos (tareas y entregas de prácticas)
  • Entregar tareas y notificaciones para seguimiento.
